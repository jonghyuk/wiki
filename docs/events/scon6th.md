# 제6회 S.CON

>S.CON 은 성공회대학교 IT 창업 동아리 S.OWL 에서 주최하는 컨퍼런스 입니다.
>각자 알고 있는 지식, 정보, 경험 등을 다수와 나누고, 새로운 사람들과 네트워킹 할 기회를 제공하는것이 S.CON의 목표이며, 동아리 내부 컨퍼런스에서 학내 컨퍼런스로 규모를 키우고 있습니다.
